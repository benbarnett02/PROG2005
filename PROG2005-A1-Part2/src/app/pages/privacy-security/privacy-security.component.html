Privacy and Security

Summary

This application stores and processes the data you supply to it. All data is stored only in memory on your device, and is not retained after your browser session completes.

The data stored and processed includes client data you provide, your IP address, and other technical details.

For more information, please read the full privacy notice below.

What data is collected?

The application collects the following data:

  • Client information you supply to the application via forms, including:
    • Name
    • Client ID
    • Email address
    • Phone number
    • Fitness program
    • Gender
    • Date of birth
    • Start/end dates of fitness program
    • Notes
    • VIP status
  • Usage data required for the application to function, including:
    • IP address
    • Browser type, viewport dimensions, device type (phone/computer/tablet)
    • Timestamps and logs of application access

Why is data collected?

Data is only collected as is necessary to provide a service to you. For example, client data is only collected in order to display the client data to you. Usage data (like IP address) is collected only to allow the application to function and continue delivering services to you.

How is data stored?

Data is stored on your device, in memory. Refreshing or closing the browser tab will clear all stored data and reset the application to it's defaults. Data is not sent outside of your device by the application.

Data is secured by not being retained.

Health Data

If you choose to store health data using this application, it is your responsibility to ensure that the health data is managed in accordance with local regulations.
